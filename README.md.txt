# 🏨 Hotel Booking Data Analysis Dashboard

A data analytics dashboard that helps hotel managers make data-driven decisions by analyzing hotel booking data. The dashboard provides insights into booking trends, cancellations, customer behavior, and revenue-related patterns through interactive visualizations.

🌐 **Live Demo:** https://hotel-dashboard-uwjj.onrender.com

---

## 📌 Project Overview

The Hotel Booking Data Analysis Dashboard transforms raw hotel booking data into meaningful insights using interactive charts and reports.

The project aims to help hotel businesses:

- Understand booking patterns
- Analyze cancellation trends
- Identify customer segments
- Improve occupancy planning
- Support marketing strategies
- Make informed business decisions

---

## 🎯 Problem Statement

Hotels often struggle with:

- High booking cancellation rates
- Seasonal fluctuations in bookings
- Limited understanding of customer demographics
- Missed marketing opportunities
- Difficulty identifying high-performing markets

This dashboard provides visual analytics to solve these challenges.

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/your-username/hotel-booking-dashboard.git
```

### Move into the project directory

```bash
cd hotel-booking-dashboard
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
python app.py
```

The application will be available at:

```
http://127.0.0.1:5000
```
